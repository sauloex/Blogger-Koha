Resumo Neste tutorial você aprenderá realizar o Backup do Banco de Dados do Koha pelo Terminal Linux

Acesse StaffLoginInstructions - Mostrar o seguinte HTML na página de login da interface da equipe e  “Clique para editar”

Apresentarei duas formas de inserir uma logo ou imagem na interface de login da equipe

1ª Usando imagem da internet

Se for usar uma imagem da internet, coloque o endereço dela dentro da tag <img> conforme a seguir.

Passo 1 - Inicialmente faremos uma pesquisa no ‘Google imagens’; (pode ser outra plataforma) por ‘logo’ (ou a imagem que você deseja inserir), em seguida é só copiar o endereço da imagem conforme abaixo:

Passo 2 - Insira no Koha o endereço copiando na tag <img> conforme exemplo a seguir:



<img src="https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcSbdFv0YZhaWVQNSV7BPtQ4Dowos7Pv5ILiLA&usqp=CAU"  alt="Exemplo logo" width= "100%" height="150">

Entenda os atributos da tag <img src="endereço da imagem" alt= “minha logo” width= "100%" height="150" >

src = este atributo é utilizado para definir o local de origem da imagem, que tanto pode ser uma pasta dentro do seu servidor ou um link externo na internet
alt = este atributo insere um texto alternativo para ser mostrado caso a imagem fique indisponível por qualquer motivo, seja por ter sido excluída ou renomeada no local de armazenamento
Width (largura) = usada para ajustar a largura da imagem
height (altura) = usada para ajustar a altura da imagem




Passo 3 - Salve as mudanças feitas. Clique em “Salvar todas staff_interface.pref preferências”


Resultado:

2ª Usando imagem dentro do servidor

Passo 1 – Salve a logo ou imagem na pasta htdocs da intranet. 

Caminho no linux: /usr/share/koha/intranet/htdocs

Para realizar este procedimento você deve ter a senha do usuário root do sistema e ter permissão para editar as pastas do Koha. Se tiver alguma dificuldade sobre isso, peça ajuda ao administrador do sistema.

Sugestão de vídeos sobre permissões no linux

Permissões de acesso a arquivos e pastas pelo terminal Linux

Comando chmod - Alterar permissões pelo terminal – Linux

Encontre a pasta htdocs em /usr/share/koha/intranet/htdocs e salve seu arquivo nela

Passo 2 - Insira no Koha o endereço copiando na tag <img> conforme exemplo a seguir: 



<img src="/Logo 02.png" alt="Logo 02" width= "100%" height="150">


Passo 3 - Salve as mudanças feitas. Clique em “Salvar todas staff_interface.pref preferências”


Resultado

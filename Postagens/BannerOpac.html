Onde no Koha: Ferramentas / Personalizações de HTML
Resumo Neste tutorial você aprenderá a inserir um banner para personalização do OPAC no Koha

Inicialmente é preciso criar uma entrada para o banner OPAC no Koha.

Para tanto acesse Ferramentas / Personalizações de HTML clique em “+ nova entrada”.

Em seguida preencha o formulário conforme imagem abaixo:

Sugestão de preenchimento

Exibir localização: opacheader
Biblioteca: Todas as bibliotecas (ou uma biblioteca específica em casa de rede de bibliotecas)
Data de publicação: DD/MM/AAAA (data a partir de quando você que o banner apareça no OPAC)
Data de validade: DD/MM/AAAA (Normalmente deixamos em branco)
Aparece na posição: preencher para ordenar se for colocar mais de um banner

Agora, apresentarei duas formas de inserir a imagem do banner no OPAC

1ª Usando um banner direto da internet

Passo 1 - Inicialmente faremos uma pesquisa no ‘Google imagens’ (pode ser outra plataforma) por ‘banners para sites’ (ou o banner que você deseja inserir), em seguida é só copiar o endereço da imagem conforme abaixo:

Passo 2 - Volte no formulário e clique em inserir imagem

Passo 3 - Preencha o campo 'fonte' com o endereço da imagem copiada no passo 2.

Em 'Descrição alternativa' coloque um nome para o seu banner (não é obrigatório).

Para largura coloque largura 1330 e Altura 150 (ou personalize como quiser).

Passo 4 - Salve as mudanças feitas.

2ª Usando imagem dentro do servidor

Passo 1 – Salve o banner na pasta htdocs do OPAC.

Caminho no linux:/usr/share/koha/opac/htdocs



Para realizar este procedimento você deve ter a senha do usuário root do sistema e ter permissão para editar as pastas do Koha. Se tiver alguma dificuldade sobre isso, peça ajuda ao administrador do sistema.

Sugestão de vídeos sobre permissões no linux

Permissões de acesso a arquivos e pastas pelo terminal Linux

Comando chmod - Alterar permissões pelo terminal – Linux

Encontre a pasta htdocs em /usr/share/koha/opac/htdocs e salve seu arquivo nela

Passo 2 - Volte no formulário e clique em inserir imagem

Passo 3 - Preencha o campo 'fonte' com o endereço da imagem do servidor "/banner 02.jpeg (lembre-se de alterar para nome do seu banner).
Em 'Descrição alternativa' coloque um nome para o seu banner (não é obrigatório). Para largura coloque 1330 e Altura 150.

Passo 5 - Salve as mudanças feitas.
